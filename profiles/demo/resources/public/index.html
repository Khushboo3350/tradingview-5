Tradingview Firefox Plugin

  1. Download manifest.json and background.js into the same folder
  2. In Firefox Open this URL: about:debugging#/runtime/this-firefox and click on [Load Temporary Add on]
  3. On tradingview.com save your chart

Technology

Uses webRequest api in firefox. https://developer.mozilla.org/en-US/docs/Mozilla/Add-ons/WebExtensions/API/webRequest/onBeforeRequest#details https://developer.mozilla.org/en-US/docs/Mozilla/Add-ons/WebExtensions/Temporary_Installation_in_Firefox
